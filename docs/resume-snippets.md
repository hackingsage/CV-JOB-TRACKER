# Resume-ready project snippets

## One-line version
CareerFlow AI, a full stack job application intelligence platform using Go, Python, React, and PostgreSQL, with JWT auth, service-based fit scoring, analytics dashboards, Dockerized local development, and CI automation.

## Stronger 3-bullet version
- Built a polyglot full stack platform using Go, Python, React, and PostgreSQL to track job applications and score resume fit against role requirements.
- Designed service boundaries between a Go API and Python analysis engine, including authentication, persistence, structured scoring output, and dashboard visualizations.
- Implemented Docker-based local development and GitHub Actions CI to improve portability, consistency, and production-readiness.

## Interview framing
Use this project to talk about:
- Why Go handled the core API
- Why Python was used for text analysis and future ML extensibility
- How you would evolve synchronous analysis into async background jobs
- What deployment and observability would look like in production
